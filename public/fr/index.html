FAQ

Basic ($24.99/an) est un VPN chiffre avec 100 Go/mois sur toutes les plateformes via l'app Outline. Pro ($99.99/an) ajoute le blocage des menaces par IA, la detection du phishing, la surveillance des connexions et un systeme de protection auto-reparateur via l'app Vizoguard — plus le VPN.

Apres votre abonnement, cliquez sur « Se connecter via Outline » sur la page de remerciement. L'app Outline s'ouvre et importe votre VPN automatiquement. Pas de copier-coller. Les utilisateurs Pro ont aussi un VPN integre dans l'app Vizoguard.

Basic : utilisez votre VPN sur n'importe quel appareil via l'app Outline (100 Go/mois partages). Pro : une licence est liee a un appareil lors de la premiere activation. Contactez support@vizoguard.com pour un transfert.

A tout moment. La protection continue jusqu'a la fin de votre periode payee. Annulez via Stripe ou contactez le support.

VPN : Mac, Windows, iOS, Android, Linux, ChromeOS. App de securite : macOS 12+ et Windows 10/11.

Les fonctionnalites de securite fonctionnent hors ligne avec les donnees de menaces en cache. La licence est verifiee toutes les 24 heures avec une periode de grace de 7 jours. Le VPN necessite une connexion Internet.
