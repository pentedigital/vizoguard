FAQ

Basic ($24.99/an) est un VPN chiffré avec 100 Go/mois sur toutes les plateformes via l'app Outline. Pro ($99.99/an) ajoute le blocage des menaces par IA, la détection du phishing, la surveillance des connexions et un système de protection auto-réparateur via l'app Vizoguard — plus le VPN.

Après votre abonnement, cliquez sur « Se connecter via Outline » sur la page de remerciement. L'app Outline s'ouvre et importe votre VPN automatiquement. Pas de copier-coller. Les utilisateurs Pro ont aussi un VPN intégré dans l'app Vizoguard.

Basic : utilisez votre VPN sur n'importe quel appareil via l'app Outline (100 Go/mois partagés). Pro : une licence est liée à un appareil lors de la première activation. Contactez support@vizoguard.com pour un transfert.

À tout moment. La protection continue jusqu'a la fin de votre période payée. Annulez via Stripe ou contactez le support.

VPN : Mac, Windows, iOS, Android, Linux, ChromeOS. App de sécurité : macOS 12+ et Windows 10/11.

Les fonctionnalités de sécurité fonctionnent hors ligne avec les données de menaces en cache. La licence est vérifiée toutes les 24 heures avec une période de grâce de 7 jours. Le VPN nécessite une connexion Internet.
