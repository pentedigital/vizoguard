Security at Vizoguard

Transparency builds trust. This page explains exactly how Vizoguard protects your data at every layer.

VPN Infrastructure

  • Powered by Outline Server, an open-source project by Jigsaw (a Google subsidiary)
  • Shadowsocks protocol with chacha20-ietf-poly1305 encryption
  • No traffic logs, no connection logs, no metadata
  • 100 GB monthly data cap per access key
  • Server location: Brazil

Desktop App Security

  • Built on Electron with contextIsolation enabled and nodeIntegration disabled
  • All API calls originate from the main process, never the renderer
  • Content Security Policy headers enforced on all screens
  • Device binding via hardware UUID (ioreg on Mac, registry on Windows)
  • License validated every 24 hours with a 7-day offline grace period

Website Security

  • HTTPS with Let's Encrypt certificates (auto-renewing)
  • HSTS with preload enabled (max-age 2 years)
  • Security headers: X-Content-Type-Options, X-Frame-Options, X-XSS-Protection, Referrer-Policy
  • No cookies, no analytics, no tracking

Payment Security

  • All payments processed by Stripe (PCI DSS Level 1 certified)
  • Vizoguard never sees or stores credit card numbers
  • Webhook verification with Stripe signatures on every event

Open Source

We believe in building on transparent foundations:

  • Outline Server — github.com/Jigsaw-Code/outline-server (Apache 2.0)
  • Vizoguard App — github.com/pentedigital/vizoguard-app (public)

Report a Security Issue

If you discover a vulnerability or have a security concern, contact support@vizoguard.com. We take every report seriously and will respond as quickly as possible.
